1. Есть ли разница между $("h1 div"); и $("h1").$("div");
- может ли привести к тому что, поиск найдёт разные элементы?
Если может - приведите пример, когда.

При поиске по селектору $("h1 div") мы найдем первый элемент,
который имеет сразу два этих параметра "h1" и "div".

При поиске по селектору $("h1").$("div") мы найдем первый элемент,
который имеет параметр "h1" и только затем, среди его детей-элементов (children),
элемент с параметром "div".
